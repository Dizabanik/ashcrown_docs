# Ambients

## <mark style="color:orange;">Overview</mark>

Ambients is a core Unlimited Adventures feature.

It is a custom-built sound system made specifically for this pack.

It creates dynamic soundscapes that react to your surroundings.

## <mark style="color:orange;">How It Reacts</mark>

Ambient audio can change based on:

* biome
* time of day
* altitude
* weather

That makes the world feel more alive as you travel.
