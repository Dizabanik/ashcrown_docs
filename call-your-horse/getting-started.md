# Getting Started

![Call Your Horse](https://wsrv.nl/?url=https%3A%2F%2Fcdn.jsdelivr.net%2Fgh%2Fjodeks-datapacks%2FAssets%40main%2Ftitles%2Fcall_your_horse.png\&n=-1)

## <mark style="color:orange;">Overview</mark>

Call Your Horse lets you bind a goat horn to a specific mount.

Using the horn teleports that mount to you, even from unloaded chunks.

By default, only the owner can bind and use the horn.

## <mark style="color:orange;">How To Bind A Horn</mark>

{% stepper %}
{% step %}
### Tame the mount

Start with a tamed mount.
{% endstep %}

{% step %}
### Give it a valid name

The name cannot contain spaces or special characters like `#`, `'`, or `_`.
{% endstep %}

{% step %}
### Hold a goat horn

Any goat horn works.
{% endstep %}

{% step %}
### Sneak and look at the mount

Keep looking at it for 4 seconds.
{% endstep %}

{% step %}
### Receive the bound horn

The horn will look enchanted and will use the mount's name.
{% endstep %}

{% step %}
### Call the mount

Use the horn at any time to teleport the mount to you.
{% endstep %}
{% endstepper %}

## <mark style="color:orange;">Reset A Horn</mark>

To unbind or reset the horn, sneak and look straight up.

## <mark style="color:orange;">Supported Mounts</mark>

By default, the pack works with:

* Horse
* Donkey
* Llama
* Mule
* Skeleton Horse
* Zombie Horse
* Trader Llama
* Camel

## <mark style="color:orange;">Rules</mark>

* The owner is the only player who can bind the horn by default.
* The owner is also the only player who can use the horn by default.
* These rules can be changed in the config.

## <mark style="color:orange;">Known Issue</mark>

If the mount is very far away, it can turn invisible after teleporting.

Rejoining the game fixes it.

{% hint style="warning" %}
This is a known vanilla-side issue and should be fixed in 1.21.
{% endhint %}
