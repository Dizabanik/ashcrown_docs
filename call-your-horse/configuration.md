# Configuration

## <mark style="color:orange;">Open The Config</mark>

Run this command:

```mcfunction
/trigger callyourhorse.config
```

## <mark style="color:orange;">Available Options</mark>

### Callable Without Owner

Choose which mounts can be called even without an owner.

### Teleport Only When Empty

Choose whether teleporting is blocked when another player is riding the mount.

This is enabled by default.

### Auto-Mount After Teleport

Choose whether you should automatically mount the called horse after it teleports.

This is disabled by default.

## <mark style="color:orange;">Default Behavior</mark>

* Owner-only binding is enabled.
* Owner-only usage is enabled.
* Teleporting with a rider is blocked.
* Auto-mount is disabled.
