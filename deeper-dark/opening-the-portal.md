# Opening the Portal

## <mark style="color:orange;">Opening a Portal</mark>

You can create a portal by using an enchanted echo shard inside the Deeper Dark.

{% stepper %}
{% step %}
### Place the key

Put an enchanted echo shard into an item frame or glow item frame at the bottom of a reinforced deepslate portal.
{% endstep %}

{% step %}
### Kill a nearby Warden

Kill a Warden within 64 blocks of the frame.
{% endstep %}

{% step %}
### Link the destination

The portal will connect to the location saved inside the echo shard.

If the shard points to a portal on the other side, it will try to light that portal too.
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
These portals are experimental. If they behave strangely, there is a gamerule that disables them.
{% endhint %}
