# Возможности

## ✨ Возможности PyroFishingPro

Ниже собраны ключевые системы плагина.

### 🐟 Кастомная рыба

**Custom Fish** позволяет создавать собственные виды рыбы.

Можно настраивать:

* биом
* день или ночь
* погоду
* высоту по Y
* регионы
* команды за поимку

### 🎨 Поддержка Custom Model Data

Рыбе можно задать уникальный внешний вид через ресурс-пак.

Это удобно для редкой, сезонной или тематической добычи.

### 📦 Deliveries

**Deliveries** — это мини-квесты на рыбалку.

Игроки ловят нужную рыбу в нужных местах и сдают её за награды.

### 🏆 Турниры

В плагине доступно **7 типов турниров**.

Можно раздавать любые призы:

* ключи
* деньги
* предметы
* кастомные награды

Также поддерживаются:

* `Boss Bar`
* уникальные эффекты
* автозапуск по времени

### 💰 Рыбный магазин

Через `/fish shop` игроки могут продавать рыбу за деньги.

Это добавляет отдельную экономическую ветку прогрессии.

### 🪄 Augments

**Augments** — это особые зачарования для удочек.

Они создаются через котёл и усиливают рыбалку. Например:

* повышают шанс на рыбу
* дают больше энтропии
* вызывают торнадо и цунами

### 🦀 Крабы

Крабы нужны для борьбы с AFK-рыбалкой.

Дополнительно они дропают полезные материалы для крафта улучшений.

### ⛔ Полу-анти-AFK система

Плагин не поощряет AFK-рыбалку.

Он использует крабов и проверки популярных рыболовных машин.

{% hint style="warning" %}
Для полной защиты от читов автор рекомендует использовать отдельный античит.
{% endhint %}

### ⚙️ Гибкая настройка

В плагине много параметров, которые можно подстроить под свой сервер.

Можно менять баланс, редкость, награды и поведение систем.

### 📈 Уровни и энтропия

У плагина есть собственные параметры прогрессии:

* **Fishing Level** — уровень рыбалки
* **Entropy** — валюта для доставок и улучшений

### 🎲 Scales

Система **Scales** позволяет рискнуть рыбой ради прибыли.

Можно как заработать, так и потерять деньги.

### 🔪 Gutting

Рыбу можно потрошить и получать бонусную энтропию.

Она нужна для создания и прокачки улучшений.

### 🧰 Настройка GUI

Меню можно кастомизировать под стиль сервера.

{% hint style="info" %}
На момент исходного описания полностью настраивались только 4 меню.
{% endhint %}

### 🗿 Totems

Тотемы добавляют уникальные пассивные эффекты и специальные механики.

### 🪱 Bait System

Можно создавать собственные наживки и покупать их во внутриигровом магазине.

Это делает рыбу более редкой и разнообразной.

### 🧩 Много дополнительной кастомизации

Плагин позволяет настраивать:

* рыбу
* наживки
* награды за поедание рыбы
* награды за продажу рыбы
* награды турниров
* нишевые параметры баланса

### 🎄 Сезонные события

В плагине есть уникальные сезонные ивенты.

Они запускаются автоматически в нужное время года и добавляют:

* особые предметы
* уникальные улучшения
* временные активности для игроков

### 🧪 Дополнительный API

Начиная с **v4.10.0**, доступен отдельный **PyroAPI**.

Он нужен разработчикам и включает события и утилиты для интеграций.
