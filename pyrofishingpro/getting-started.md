# Getting Started

## 🎣 PyroFishingPro

PyroFishingPro превращает обычную рыбалку в отдельную игровую систему.

Здесь есть кастомная рыба, турниры, доставки, наживки, улучшения удочек и собственная прогрессия.

![](https://i.imgur.com/B9otCFt.png)

![](https://i.imgur.com/pMiL3TB.png)

### 🛟 Поддержка

Если нужен саппорт или хотите следить за обновлениями, используйте Discord автора:

* [Discord-сервер](https://discord.gg/zNtFN3m)

{% hint style="warning" %}
Начиная с версии **4.6.0**, плагин требует установленный **PyroLib**.

Скачать библиотеку можно здесь: [PyroLib](https://www.spigotmc.org/resources/pyrolib.98255/)
{% endhint %}

### ✅ Что даёт плагин

* кастомную рыбу с собственными условиями поимки
* внутриигровой магазин и продажу рыбы
* турниры по рыбалке
* мини-квесты через систему доставок
* энтропию, уровни рыбалки и улучшения
* наживки, тотемы, крабов и сезонные ивенты

### 🚀 Быстрый старт

{% stepper %}
{% step %}
### Установите зависимости

Поставьте **PyroLib**, затем сам **PyroFishingPro**.
{% endstep %}

{% step %}
### Перезапустите сервер

После запуска проверьте, что плагин загрузился без ошибок.
{% endstep %}

{% step %}
### Откройте меню рыбалки

Используйте `/fish menu`, чтобы открыть главное меню плагина.
{% endstep %}

{% step %}
### Начните прогрессию

Ловите рыбу, копите энтропию, выполняйте доставки и открывайте улучшения.
{% endstep %}
{% endstepper %}

### 📌 Важные ограничения

* Поддерживаются только версии **Minecraft 1.14+**.
* Для старых версий поддержка не заявлена.
* Часть админских прав и настроек хранится в `config.yml`.
