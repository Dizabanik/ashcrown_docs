# Getting Started

## 🎣 Быстрый старт

PyroFishingPro превращает рыбалку в отдельную ветку прогрессии.

Вы ловите редкую рыбу, копите энтропию, выполняете доставки и усиливаете удочку.

![](https://i.imgur.com/B9otCFt.png)

![](https://i.imgur.com/pMiL3TB.png)

### Что здесь есть

* рыба разной редкости
* магазин и продажа улова
* доставки и турниры
* улучшения удочки
* наживки, тотемы и сезонные события

### С чего начать

{% stepper %}
{% step %}
### Откройте главное меню

Используйте `/fish menu`.
{% endstep %}

{% step %}
### Поймайте первую рыбу

Начните с обычной рыбалки и проверьте, какие виды доступны в вашем мире.
{% endstep %}

{% step %}
### Тратьте улов с пользой

Продавайте рыбу через `/fish shop` или сохраняйте её для доставок.
{% endstep %}

{% step %}
### Развивайте прогрессию

Копите энтропию, открывайте улучшения и участвуйте в турнирах.
{% endstep %}
{% endstepper %}

### На что смотреть в первую очередь

* `/fish codex` — посмотреть прогресс и коллекцию
* `/fish deliveries` — брать задания на рыбу
* `/fish augment` — усиливать удочку
* `/fish bait` — покупать и использовать наживки

{% hint style="info" %}
Если какая-то команда не работает, значит сервер мог отключить её или открыть позже по прогрессии.
{% endhint %}
