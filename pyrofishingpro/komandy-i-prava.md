# Команды и права

## ⌨️ Команды PyroFishingPro

{% hint style="info" %}
Аргументы в `<угловых скобках>` обязательные. Аргументы в `[квадратных]` необязательные.
{% endhint %}

### 🎣 Базовые команды

| Команда                | Описание                                   |
| ---------------------- | ------------------------------------------ |
| `/fish`                | Открыть базовую справку по плагину.        |
| `/fish menu`           | Открыть главное меню рыбалки.              |
| `/fish codex`          | Открыть кодекс.                            |
| `/fish shop`           | Открыть магазин рыбы.                      |
| `/fish scales`         | Открыть интерфейс Scales.                  |
| `/fish bag`            | Открыть сумку для рыбы.                    |
| `/fish gut`            | Открыть меню потрошения.                   |
| `/fish deliveries`     | Открыть меню доставок.                     |
| `/fish augment`        | Открыть меню улучшений удочки.             |
| `/fish augments`       | Показать все улучшения для рыбалки в игре. |
| `/fish entropy`        | Посмотреть текущий баланс энтропии.        |
| `/fish boosters`       | Посмотреть активные бустеры.               |
| `/fish bait`           | Открыть магазин наживок.                   |
| `/fish removedelivery` | Удалить текущую доставку.                  |

### 🛠️ Админские команды

| Команда                                   | Описание                             |
| ----------------------------------------- | ------------------------------------ |
| `/fish reload`                            | Перезагрузить плагин и конфигурацию. |
| `/fish give <player> <fishName> <amount>` | Выдать игроку конкретную рыбу.       |
| `/fish addentropy <player> <amount>`      | Выдать игроку энтропию.              |
| `/fish addxp <player> <amount>`           | Выдать игроку опыт рыбалки.          |

### 🏆 Команды турниров

| Команда                             | Описание                               |
| ----------------------------------- | -------------------------------------- |
| `/ft`                               | Базовая команда турниров.              |
| `/ft info`                          | Показать информацию о текущем турнире. |
| `/ft types`                         | Показать доступные типы турниров.      |
| `/ft time`                          | Показать время до следующего турнира.  |
| `/ft start <tournamentType> [time]` | Запустить турнир указанного типа.      |
| `/ft stop`                          | Остановить текущий турнир.             |
| `/ft remove <player>`               | Удалить игрока из турнира.             |

{% hint style="warning" %}
Для управления турнирами нужен доступ `fish.tournaments`.
{% endhint %}

### 🔐 Права доступа

Ниже перечислены права, которые были явно указаны в исходном описании.

| Право                      | Назначение                                                         |
| -------------------------- | ------------------------------------------------------------------ |
| `fish.totems.break.others` | Позволяет ломать тотемы других игроков.                            |
| `fish.blockcodex`          | Запрещает игроку становиться первым, кто поймал определённую рыбу. |
| `fish.tournaments`         | Даёт доступ к управлению турнирами.                                |

{% hint style="info" %}
Не все права перечислены здесь. Часть разрешений задаётся в `config.yml`.
{% endhint %}
