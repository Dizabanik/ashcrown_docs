# Условия использования

## 📜 Условия использования

Ниже приведены условия из исходного описания ресурса.

### 🚫 Что запрещено

* Нельзя распространять плагин на других сайтах или площадках без прямого разрешения автора.
* Нельзя использовать ресурс для личной выгоды без явного разрешения автора.
* Нельзя выкладывать баг-репорты в разделе отзывов.
* Нельзя декомпилировать плагин без прямого разрешения автора.
* Нельзя делиться ресурсом с другими людьми.
* Нельзя использовать плагин без покупки.

### 💸 Возвраты

Возвраты не предусмотрены ни при каких обстоятельствах.

Перед покупкой убедитесь, что плагин вам действительно нужен.

### 🧾 Лицензия

Одна покупка предназначена для **одного сервера или одной сети**.

Использование на нескольких независимых серверах не разрешено.

### 🛠️ Если нашли баг

Свяжитесь с автором напрямую.

Указанный контакт в исходном описании: **pyrotempus**.

### 📌 Дополнительно

* Доступ к плагину открывается сразу после покупки.
* Поддерживаются только версии **1.14 и выше**.
* Для более старых версий поддержка не предоставляется.

{% endhint %}
