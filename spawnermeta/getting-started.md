# Getting Started

## <mark style="color:orange;">Overview</mark>

SpawnerMeta adds a custom spawner system.

You can buy spawners, upgrade them, move them, and change their mob type.

{% hint style="info" %}
Spawner purchases and upgrades use **Experience Points**, not experience levels.

Some servers also allow money as a second payment option.
{% endhint %}

## <mark style="color:orange;">Getting Your First Spawner</mark>

{% stepper %}
{% step %}
### Open the shop

Run `/spawnershop` or `/spawners shop`.
{% endstep %}

{% step %}
### Buy a spawner

Pick the mob spawner you want and pay with XP.
{% endstep %}

{% step %}
### Place it down

Put the spawner where you want your farm to run.
{% endstep %}

{% step %}
### Open upgrades

Right-click the placed spawner to open its upgrade menu.
{% endstep %}
{% endstepper %}

## <mark style="color:orange;">Upgrading Spawners</mark>

Placed spawners can be upgraded to improve farm output.

Natural spawners can also be upgraded.

### Upgrade stats

* **Delay** reduces the time between spawn cycles.
* **Amount** increases the number of mobs per cycle.
* **Range** increases the valid spawn radius around the spawner.

{% hint style="success" %}
All upgrade purchases are made through the spawner menu.
{% endhint %}

## <mark style="color:orange;">Moving And Modifying Spawners</mark>

* **Pick up a spawner:** left-click it to mine and collect it.
* **Keep progress:** upgrades and remaining charges stay on the spawner after pickup.
* **Change mob type:** use the correct spawn egg on the spawner.
* **Read holograms:** placed spawners can show a floating label with their mob type.

{% hint style="info" %}
You do not lose your upgrade progress when moving a spawner.
{% endhint %}

## <mark style="color:orange;">Player Commands</mark>

| Command             | Use                                               |
| ------------------- | ------------------------------------------------- |
| `/spawnershop`      | Open the shop and buy new spawners.               |
| `/spawnerview`      | View your spawners and their current stats.       |
| `/spawnerlocations` | Show the locations of your placed spawners.       |
| `/spawnertrust`     | Manage trusted players for your spawners.         |
| `/spawnerdrops`     | Recover spawner drops if your inventory was full. |

## <mark style="color:orange;">Quick Tips</mark>

* Upgrade **Delay** first for faster output.
* Use **Amount** upgrades when you need denser spawn cycles.
* Use **Range** upgrades only if your farm has enough free space.
* Check `/spawnerlocations` before rebuilding or moving a base.
