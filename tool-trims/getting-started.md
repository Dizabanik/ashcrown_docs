# Getting Started

## <mark style="color:orange;">Content</mark>

This is the Tool Trims section.

Here you will find how tool trim templates work, where to find them, which materials can be used, and which advancements are tied to the pack.

Tool Trims adds smithing templates for weapons and tools. It lets you trim them in a Smithing Table with 11 different materials and more than `1700` possible combinations.
