# Useful Commands

## 📖 Команды для игроков

{% hint style="info" %}
На этой странице собраны все доступные команды для игроков. Аргументы в `<угловых скобках>` — обязательные, в `[квадратных]` — необязательные.
{% endhint %}

***

## ⚙️ Быстрые основные команды

Здесь собраны все основные команды сервера. Большинство из них также доступны через главное меню `/menu`.

***

### 🗺️ Навигация и меню

| Команда    | Описание                                                                                           |
| ---------- | -------------------------------------------------------------------------------------------------- |
| `/menu`    | Открыть главное меню — доступ к профилю, настройкам, аксессуарам, наградам и всем системам сервера |
| `/profile` | Открыть профиль персонажа — статистика, активные квесты, время в игре                              |
| `/gear`    | Быстрый доступ к меню аксессуаров без открытия главного меню                                       |
| `/skills`  | Открыть меню навыков — прокачка характеристик, просмотр прогресса                                  |

***

### 🏆 Прогрессия и награды

| Команда   | Описание                                                                                                                               |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| `/daily`  | Получить ежедневную награду. Цикл — 14 дней, каждый день — новая награда. На 7-й и 14-й день — особые награды                          |
| `/guilds` | Меню трёх фракций сервера — **Жнецы**, **Глубинники**, **Пастыри**. Каждая фракция предлагает уникальные квесты и награды за репутацию |

***

### ⚔️ Гильдии игроков

Гильдии — отдельная система объединения игроков. Подробнее: [Гильдии](useful-commands.md#guilds-gildii)

Основная команда для гильдий игроков: `/guild`  или `/g` .

***

### 🗃️ Спавнеры

Спавнеры можно улучшать, заменять и добывать без потери.

| Действие                                    | Описание                                                                                   |
| ------------------------------------------- | ------------------------------------------------------------------------------------------ |
| **ПКМ по спавнеру**                         | Открыть меню улучшений спавнера — прокачка дальности, задержки и количества мобов          |
| **Кирка с «Шёлковым касанием» по спавнеру** | Добыть спавнер — он автоматически попадёт в инвентарь. Убедитесь, что есть свободное место |
| **Shift + ПКМ по спавнеру с яйцом призыва** | Заменить тип моба в спавнере на моба из яйца в руке                                        |

{% hint style="warning" %}
Если инвентарь заполнен в момент добычи спавнера — он не попадёт к вам. Освободите место заранее.
{% endhint %}

### 💀 Смерть и надгробия

При смерти ваш инвентарь сохраняется в надгробии на месте гибели.

| Команда       | Описание                                                |
| ------------- | ------------------------------------------------------- |
| `/grave list` | Показать список ваших активных надгробий с координатами |

{% hint style="warning" %}
Надгробия существуют ограниченное время. Не откладывайте возврат за вещами.
{% endhint %}

### ⚔️ AuraSkills — Навыки и характеристики

AuraSkills — система прокачки навыков. Чем больше вы занимаетесь определённым делом (рубите деревья, копаете, сражаетесь), тем выше растёт соответствующий навык, открывая новые способности и усиливая персонажа.

| Команда           | Описание                                                                             |
| ----------------- | ------------------------------------------------------------------------------------ |
| `/skills`         | Открыть меню навыков — здесь видно весь ваш прогресс, уровни и доступные способности |
| `/stats`          | Показать все характеристики персонажа: здоровье, урон, защиту и прочее               |
| `/sk top [навык]` | Таблица лидеров по уровню навыков — посмотрите, кто прокачался сильнее всех          |
| `/sk rank`        | Узнать свой текущий ранг среди всех игроков сервера                                  |

### 🏰 Guilds — Гильдии

Гильдии позволяют объединяться с другими игроками, строить общую базу, пользоваться банком, хранилищем и дипломатией. Ниже собраны все основные команды по категориям.

#### ⭐ Самые необходимые команды

| Команда                         | Описание                                          |
| ------------------------------- | ------------------------------------------------- |
| `/guild create <имя> [префикс]` | Создать новую гильдию                             |
| `/guild invite <игрок>`         | Пригласить игрока в вашу гильдию                  |
| `/guild accept <название>`      | Принять приглашение на вступление в гильдию       |
| `/guild leave`                  | Покинуть текущую гильдию                          |
| `/guild chat`                   | Включить или выключить приватный чат гильдии      |
| `/guild home`                   | Телепортироваться на базу или домой вашей гильдии |

#### 🖥️ Интерфейс (GUI)

| Команда          | Описание                                       |
| ---------------- | ---------------------------------------------- |
| `/guild buff`    | Открыть меню баффов или улучшений гильдии      |
| `/guild list`    | Открыть список всех существующих гильдий       |
| `/guild members` | Открыть список участников вашей гильдии        |
| `/guild vault`   | Открыть виртуальное хранилище гильдии          |
| `/guild info`    | Открыть меню с подробной информацией о гильдии |

#### 👑 Управление гильдией

| Команда                   | Описание                                             |
| ------------------------- | ---------------------------------------------------- |
| `/guild delete`           | Удалить текущую гильдию. Доступно только лидеру      |
| `/guild kick <игрок>`     | Выгнать игрока из гильдии                            |
| `/guild prefix <префикс>` | Установить новый префикс гильдии                     |
| `/guild rename <имя>`     | Изменить название гильдии                            |
| `/guild status`           | Сменить статус гильдии: открытая или по приглашениям |
| `/guild transfer <игрок>` | Передать лидерство другому участнику                 |
| `/guild upgrade`          | Повысить уровень или тир вашей гильдии               |

#### 👥 Участники

| Команда                     | Описание                                          |
| --------------------------- | ------------------------------------------------- |
| `/guild check`              | Проверить, есть ли у вас ожидающие приглашения    |
| `/guild decline <название>` | Отклонить приглашение в гильдию                   |
| `/guild demote <игрок>`     | Понизить ранг игрока в вашей гильдии              |
| `/guild promote <игрок>`    | Повысить ранг игрока в вашей гильдии              |
| `/guild request <гильдия>`  | Отправить запрос на вступление в закрытую гильдию |

#### 🤝 Альянсы (Союзники)

| Команда                         | Описание                                |
| ------------------------------- | --------------------------------------- |
| `/guild ally add <гильдия>`     | Отправить запрос на союз другой гильдии |
| `/guild ally accept <гильдия>`  | Принять входящий запрос на союз         |
| `/guild ally decline <гильдия>` | Отклонить запрос на союз                |
| `/guild ally list`              | Показать список всех ваших союзников    |
| `/guild ally remove <гильдия>`  | Разорвать союз с указанной гильдией     |

#### 💰 Банк гильдии

Каждая гильдия имеет общий банк. Участники могут пополнять его для развития гильдии.

| Команда                        | Описание                                        |
| ------------------------------ | ----------------------------------------------- |
| `/guild bank balance`          | Показать текущий баланс банка гильдии           |
| `/guild bank deposit <сумма>`  | Перевести деньги с личного счёта в банк гильдии |
| `/guild bank withdraw <сумма>` | Снять деньги из банка гильдии. Зависит от прав  |

#### 🚩 Приваты (Клеймы)

Эта функция использует WorldGuard для захвата и защиты земли под нужды гильдии.

| Команда          | Описание                                         |
| ---------------- | ------------------------------------------------ |
| `/guild claim`   | Создать защищённую территорию гильдии вокруг вас |
| `/guild unclaim` | Удалить текущий приват вашей гильдии             |

#### 🏠 Точки дома

| Команда          | Описание                                       |
| ---------------- | ---------------------------------------------- |
| `/guild sethome` | Установить точку дома гильдии на текущем месте |
| `/guild home`    | Телепортироваться на базу гильдии              |
| `/guild delhome` | Удалить точку дома вашей гильдии               |

#### 🎫 Коды приглашений

Можно создать одноразовый или многоразовый код. По нему игроки вступают без личного приглашения.

| Команда                                         | Описание                                  |
| ----------------------------------------------- | ----------------------------------------- |
| `/guild code create [количество_использований]` | Создать код для вступления в вашу гильдию |
| `/guild code delete <код>`                      | Удалить существующий код приглашения      |
| `/guild code info <код>`                        | Показать подробную информацию о коде      |
| `/guild code list`                              | Вывести список всех активных кодов        |
| `/guild code redeem <код>`                      | Использовать код для вступления в гильдию |

#### ⚔️ Войны гильдий

| Команда                          | Описание                                      |
| -------------------------------- | --------------------------------------------- |
| `/guild war challenge <гильдия>` | Бросить вызов другой гильдии на войну         |
| `/guild war accept`              | Принять брошенный вам вызов на войну          |
| `/guild war deny`                | Отклонить вызов на войну                      |
| `/guild war join`                | Присоединиться к активной войне вашей гильдии |

#### 📜 Сообщение дня (MOTD)

| Команда                       | Описание                                     |
| ----------------------------- | -------------------------------------------- |
| `/guild motd`                 | Показать текущее сообщение дня вашей гильдии |
| `/guild motd set <сообщение>` | Установить новое сообщение дня               |
| `/guild motd remove`          | Удалить сообщение дня                        |

{% hint style="warning" %}
Удаление гильдии через `/guild delete` необратимо. Хранилище, банк и прогресс будут потеряны.
{% endhint %}

### 🌾 CustomCrops — Пользовательские культуры

CustomCrops добавляет расширенную систему земледелия: новые растения, удобрения, горшки и поливочные механики. Большинство взаимодействий происходит через правый клик по блокам, но есть и несколько команд.

| Команда          | Описание                                                                           |
| ---------------- | ---------------------------------------------------------------------------------- |
| `/ccrops`        | Открыть главное меню плагина — здесь можно посмотреть доступные культуры и рецепты |
| `/ccrops season` | Узнать текущий сезон на сервере — от него зависит, какие культуры будут расти      |
| `/ccrops plants` | Просмотреть справочник всех доступных растений и условий их роста                  |

{% hint style="info" %}
**Как работает земледелие:** посадите семена в специальный горшок или грядку, своевременно поливайте и используйте удобрения для ускорения роста. Урожай правым кликом не ломается — он сбрасывает плоды и сразу перерастает обратно.
{% endhint %}

***

### ⚰️ AxGraves — Могилы

AxGraves автоматически создаёт могилу на месте вашей гибели. Все предметы из инвентаря сохраняются в ней — вы не потеряете вещи, если успеете добраться до могилы вовремя.

| Команда        | Описание                                                                    |
| -------------- | --------------------------------------------------------------------------- |
| `/graves`      | Открыть список всех ваших активных могил с координатами и временем создания |
| `/graves list` | То же самое — показать все ваши могилы                                      |

{% hint style="warning" %}
Могила существует ограниченное время — успейте добраться до неё, пока не истёк таймер, иначе вещи выпадут на землю и могут исчезнуть.
{% endhint %}

***

### 🧿 SpawnerMeta — Информация о спавнерах

SpawnerMeta позволяет узнавать подробную информацию о спавнерах мобов, а также в некоторых случаях изменять их параметры. [Больше о спавнерах сдесь.](spawnermeta/getting-started.md)

{% hint style="info" %}
ПКМ по спавнеру также работает.
{% endhint %}

| Команда        | Описание                                                                                                 |
| -------------- | -------------------------------------------------------------------------------------------------------- |
| `/spawnermeta` | Открыть главное меню плагина                                                                             |
| `/sm info`     | Посмотреть информацию о спавнере, на который вы смотрите: тип моба, задержку спавна, дальность активации |
| `/sm list`     | Показать список всех спавнеров в округе, к которым у вас есть доступ                                     |
| `/sm stack`    | Объединить несколько одинаковых спавнеров в один мощный (если это разрешено правилами сервера)           |

{% hint style="info" %}
Чтобы посмотреть информацию о спавнере, встаньте рядом с ним и выполните `/sm info` — плагин автоматически определит ближайший спавнер.
{% endhint %}

## 🎣 EvenMoreFish — Рыбалка и соревнования

EvenMoreFish — расширенная система рыбалки с редкими рыбами, наживками, магазином и регулярными соревнованиями среди игроков. Просто возьмите удочку и закидывайте!

| Команда                   | Описание                                                                                         |
| ------------------------- | ------------------------------------------------------------------------------------------------ |
| `/emf`                    | Показать справочное сообщение плагина                                                            |
| `/emf help`               | Подробная справка по всем командам                                                               |
| `/emf gui`                | Открыть главное меню EvenMoreFish                                                                |
| `/emf next`               | Узнать, когда начнётся следующее рыболовное соревнование                                         |
| `/emf top`                | Таблица лидеров текущего соревнования — посмотрите, кто впереди                                  |
| `/emf toggle`             | Включить или отключить получение наград за рыбалку                                               |
| `/emf shop [игрок]`       | Открыть магазин, где можно продать пойманную рыбу                                                |
| `/emf sellall`            | Мгновенно продать всю рыбу из инвентаря по текущим ценам                                         |
| `/emf applybaits`         | Открыть меню применения наживок на удочку                                                        |
| `/emf journal [редкость]` | Открыть журнал коллекции рыб — здесь хранится всё, что вы поймали. Можно фильтровать по редкости |

{% hint style="info" %}
**Наживки** значительно повышают шанс поймать редкую рыбу. Зайдите в `/emf applybaits` перед серьёзной рыбалкой — это стоит того.
{% endhint %}
