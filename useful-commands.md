# Useful Commands

## 📖 Команды для игроков

{% hint style="info" %}
На этой странице собраны все доступные команды для игроков. Аргументы в `<угловых скобках>` — обязательные, в `[квадратных]` — необязательные.
{% endhint %}

### ⚔️ AuraSkills — Навыки и характеристики

AuraSkills — система прокачки навыков. Чем больше вы занимаетесь определённым делом (рубите деревья, копаете, сражаетесь), тем выше растёт соответствующий навык, открывая новые способности и усиливая персонажа.

| Команда           | Описание                                                                             |
| ----------------- | ------------------------------------------------------------------------------------ |
| `/skills`         | Открыть меню навыков — здесь видно весь ваш прогресс, уровни и доступные способности |
| `/stats`          | Показать все характеристики персонажа: здоровье, урон, защиту и прочее               |
| `/claimitems`     | Получить награды за достижения в навыках, которые накопились и ждут получения        |
| `/sk top [навык]` | Таблица лидеров по уровню навыков — посмотрите, кто прокачался сильнее всех          |
| `/sk rank`        | Узнать свой текущий ранг среди всех игроков сервера                                  |

{% hint style="info" %}
Открывайте `/claimitems` регулярно — награды за прокачку навыков не начисляются автоматически, их нужно забирать вручную.
{% endhint %}

### 🏰 Guilds — Гильдии

Гильдии позволяют объединяться с другими игроками, строить общий дом, делиться ресурсами и вместе развиваться. Гильдия прокачивается, открывая новые возможности — больше участников, буферы, хранилище.

#### Основное

| Команда                          | Описание                                                         |
| -------------------------------- | ---------------------------------------------------------------- |
| `/guild create <название>`       | Создать новую гильдию с выбранным названием                      |
| `/guild list`                    | Просмотреть список всех существующих гильдий на сервере          |
| `/guild info [название]`         | Посмотреть информацию о своей гильдии или о любой другой         |
| `/guild join <название>`         | Вступить в гильдию (если она открыта для всех)                   |
| `/guild leave`                   | Покинуть текущую гильдию                                         |
| `/guild delete`                  | Полностью удалить гильдию (только для лидера!)                   |
| `/guild rename <новое название>` | Переименовать гильдию                                            |
| `/guild prefix <префикс>`        | Установить тег гильдии, который будет отображаться рядом с ником |

#### Участники и управление

| Команда                   | Описание                                 |
| ------------------------- | ---------------------------------------- |
| `/guild invite <игрок>`   | Пригласить игрока в гильдию              |
| `/guild accept`           | Принять приглашение в гильдию            |
| `/guild decline`          | Отклонить приглашение в гильдию          |
| `/guild kick <игрок>`     | Исключить игрока из гильдии              |
| `/guild promote <игрок>`  | Повысить роль игрока внутри гильдии      |
| `/guild demote <игрок>`   | Понизить роль игрока внутри гильдии      |
| `/guild transfer <игрок>` | Передать звание лидера другому участнику |

#### Общение и дом

| Команда               | Описание                                                                      |
| --------------------- | ----------------------------------------------------------------------------- |
| `/guild chat`         | Переключить чат в режим гильдии — все сообщения будут видны только участникам |
| `/guild motd <текст>` | Установить сообщение дня для гильдии (Message of the Day)                     |
| `/guild sethome`      | Установить точку дома гильдии на месте, где вы стоите                         |
| `/guild home`         | Телепортироваться к дому гильдии                                              |

#### Экономика и развитие

| Команда                   | Описание                                                                    |
| ------------------------- | --------------------------------------------------------------------------- |
| `/guild deposit <сумма>`  | Пополнить казну гильдии своими деньгами                                     |
| `/guild withdraw <сумма>` | Снять деньги из казны гильдии (только для офицеров и лидера)                |
| `/guild upgrade`          | Улучшить уровень гильдии за деньги — открывает больше слотов и возможностей |
| `/guild vault`            | Открыть общее хранилище гильдии                                             |
| `/guild buff`             | Активировать временный бонус для всех участников гильдии                    |
| `/guild claimland`        | Застолбить чанк под защиту гильдии                                          |
| `/guild unclaimland`      | Снять защиту с чанка                                                        |

#### Дипломатия

| Команда                        | Описание                       |
| ------------------------------ | ------------------------------ |
| `/guild ally add <гильдия>`    | Предложить союз другой гильдии |
| `/guild ally remove <гильдия>` | Разорвать союз с гильдией      |
| `/guild war declare <гильдия>` | Объявить войну другой гильдии  |

{% hint style="warning" %}
Удаление гильдии через `/guild delete` необратимо — все данные, хранилище и прокачка будут потеряны навсегда.
{% endhint %}

### 🌾 CustomCrops — Пользовательские культуры

CustomCrops добавляет расширенную систему земледелия: новые растения, удобрения, горшки и поливочные механики. Большинство взаимодействий происходит через правый клик по блокам, но есть и несколько команд.

| Команда          | Описание                                                                           |
| ---------------- | ---------------------------------------------------------------------------------- |
| `/ccrops`        | Открыть главное меню плагина — здесь можно посмотреть доступные культуры и рецепты |
| `/ccrops season` | Узнать текущий сезон на сервере — от него зависит, какие культуры будут расти      |
| `/ccrops plants` | Просмотреть справочник всех доступных растений и условий их роста                  |

{% hint style="info" %}
**Как работает земледелие:** посадите семена в специальный горшок или грядку, своевременно поливайте и используйте удобрения для ускорения роста. Урожай правым кликом не ломается — он сбрасывает плоды и сразу перерастает обратно.
{% endhint %}

### ⚰️ AxGraves — Могилы

AxGraves автоматически создаёт могилу на месте вашей гибели. Все предметы из инвентаря сохраняются в ней — вы не потеряете вещи, если успеете добраться до могилы вовремя.

| Команда              | Описание                                                                       |
| -------------------- | ------------------------------------------------------------------------------ |
| `/graves`            | Открыть список всех ваших активных могил с координатами и временем создания    |
| `/graves list`       | То же самое — показать все ваши могилы                                         |
| `/graves tp <id>`    | Телепортироваться к могиле по её номеру (если эта функция включена на сервере) |
| `/graves fetch <id>` | Забрать содержимое могилы к себе в инвентарь без телепортации (если разрешено) |

{% hint style="warning" %}
Могила существует ограниченное время — успейте добраться до неё, пока не истёк таймер, иначе вещи выпадут на землю и могут исчезнуть.
{% endhint %}

### 🧿 SpawnerMeta — Информация о спавнерах

SpawnerMeta позволяет узнавать подробную информацию о спавнерах мобов, а также в некоторых случаях изменять их параметры.

| Команда        | Описание                                                                                                 |
| -------------- | -------------------------------------------------------------------------------------------------------- |
| `/spawnermeta` | Открыть главное меню плагина                                                                             |
| `/sm info`     | Посмотреть информацию о спавнере, на который вы смотрите: тип моба, задержку спавна, дальность активации |
| `/sm list`     | Показать список всех спавнеров в округе, к которым у вас есть доступ                                     |
| `/sm stack`    | Объединить несколько одинаковых спавнеров в один мощный (если это разрешено правилами сервера)           |

{% hint style="info" %}
Чтобы посмотреть информацию о спавнере, встаньте рядом с ним и выполните `/sm info` — плагин автоматически определит ближайший спавнер.
{% endhint %}

### 💬 Быстрая шпаргалка

{% tabs %}
{% tab title="🆕 Новый игрок" %}
Первые шаги на сервере:

1. `/skills` — посмотрите на систему навыков и поймите, что качать
2. `/guild list` — найдите гильдию по вкусу и вступите, это сильно упростит начало
3. `/stats` — проверьте свои характеристики после первых уровней навыков
4. `/graves` — на случай гибели, знайте, что ваши вещи в безопасности
{% endtab %}

{% tab title="⚔️ Боец" %}
Полезные команды для PvE и PvP:

* `/stats` — следите за уровнем атаки, защиты и здоровья
* `/skills` → раздел боевых навыков — смотрите доступные способности
* `/claimitems` — забирайте награды за прокачку боевых навыков
* `/guild buff` — активируйте боевой бонус перед рейдом
{% endtab %}

{% tab title="🌿 Фермер" %}
Полезные команды для сельского хозяйства:

* `/ccrops` — справочник культур и рецептов
* `/ccrops season` — не забывайте про актуальный сезон
* `/skills` → навык Земледелия — смотрите прогресс и разблокируйте бонусы
* `/claimitems` — получайте награды за уровни Земледелия
{% endtab %}

{% tab title="🏰 Лидер гильдии" %}
Ключевые команды для управления гильдией:

* `/guild upgrade` — прокачивайте гильдию при первой возможности
* `/guild vault` — следите за общим хранилищем
* `/guild motd <текст>` — информируйте участников через сообщение дня
* `/guild claimland` — защищайте территорию базы
* `/guild ally add <гильдия>` — выстраивайте союзы с соседями
{% endtab %}
{% endtabs %}
